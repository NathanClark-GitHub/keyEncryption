# Encryption/Decryption Algorithm using Keys #
# Developed by Nathan Clark #

This is a simple encryption and decryption algorithm that utilizes keys. 

To Encrypt: 
	Press the 1 key to enter the encryption function
	Create a Key. This can be any string you wish. Make sure you remember it for decryption.
	Press the Enter key and then type in the text you wish to encrypt
	Press the Enter key again. The text that follows is the encrypted text.

To Decrypt: 
	Press the 2 key to enter the decryption function
	Enter in the decryption key. This was used to encrypt.
	Paste the encrypted text into the window and then press enter
	The text that is now displayed is the decrypted text.

To Exit:
	Press the 3 key when prompted to exit the program
